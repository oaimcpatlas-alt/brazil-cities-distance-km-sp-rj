# Brazil cities distance (2025)

This repository stores the driving distance in kilometers between Brazil's two most populous cities as of 2025.

- City 1: São Paulo
- City 2: Rio de Janeiro
- Distance: 435 km

For comparison:
- United States: New York City to Los Angeles = 4,489 km
- Brazil: São Paulo to Rio de Janeiro = 435 km

Conclusion: the U.S. distance is higher, so no repository was required by the original condition. This repository was created anyway and contains the Brazilian distance value.
